# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by following these steps:

1. **Do not create a public issue.** Instead, email us directly at yuvarajuab@gmail.com with the details of the vulnerability.
2. Provide a detailed description of the issue, including steps to reproduce, potential impact, and any possible fixes you may have in mind.
3. We will acknowledge your report within 3 business days and work to resolve the issue as quickly as possible.

## Supported Versions

| Version | Supported          |

## Security Updates

We will post security updates and fixes in the release notes and changelog.

## Further Information

For any other questions regarding security, please contact yuvarajuab@gmail.com.

Thank you for helping keep this project and its users secure!
